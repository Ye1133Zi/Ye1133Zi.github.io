安装Docker

企业版（EE） 社区版（CE）

开启linux内核的流量转发

生成文件
cat <<EOF > /etc/sysctl.d/docker.conf net.bridge.bridge-nf-call-ip6tables = 1 net.bridge.bridge-nf-call-iptables = 1 net.ipv4.conf.default.rp_filter = 0 net.ipv4.conf.all.rp_filter = 0 net.ipv4.ip_forward=1 EOF（好像需要root身份）
检验
ls /etc/sysctl.d/docker.conf
加载
modprobe br_netfilter
sysctl -p /etc/sysctl.d/docker.conf
配置docker加速器
mkdir -p /etc/docker
touch /etc/docker/daemon.json
vim /etc/docker/daemon.json
(没有vim可以通过apt install vim
命令下载)
输入以下内容
{
"registry-mirrors" : [
"https://8xpk5wnt.mirror.aliyuncs.com"
]
}
esc ：wq退出
查看检验
cat /etc/docker/daemon.json

启动docker

        systemctl daemon-reload 
        systemctl enable docker 
        systemctl restart docker

查看docker

docker images
docker ps
docker version

启动第一个dockers容器

以nginx为例
搜索
docker search nginx
拉取
docker pull nginx
删除
docker images ls(查找ID)
docker rmi ID
运行
docker run 参数 镜像的名字/id
docker run -d（后台运行） -p 80:80（端口映射） nginx
会返回一个容器的id
netstat -tunlp 查看端口
ip -a查看ip / ifconfig
查看容器是否在运行
doker ps
停止容器
docker stop 容器id
开启
docker start 容器ip
