linux命令

目录：

-l 详尽信息
pwd （print working directory）
~ 代表当前用户主目录
. 代表当前用户目录
.. 上一级目录
mkdir（make directory）创建目录 ，多级目录使用-p
rmdir （remove directory）删除目录（空）
rm （remove） -r（recursive递归）f（force）
cp A->B(将A复制到B)
mv 移动或者重命名

文件：

文件权限
-owner
-r（read）
-w（write）
-x （excut）
查看权限：ls -l
九个字符：
rw-自己 r–同组 r–别人
修改权限：chmod（chnage file mod）
chmod o+w name.txt
o表示别人 +w表示增加write权限
a（all）所有人 u自己
修改属主
chown（—R递归）
opt目录可以放一些公共的东西
先名后目录

归档：

tar （tape archive）打包成Tape包
例：tar -cvf example.tar exmple
其中
c，create，创建档案
v，verbose，显示详情
f，file
也可以多个文件打包成一个
解开tar -xvf （x，extract）后面加-C可以指定文件
tar -czvf example.tar.gz example

软链接：

ln （link） -s（soft）本体 链接名

对用户操作：

useradd
sudo（管理员身份运行，需要sudoer）
-m 在/home下
sudo useradd -m name 创建
sudo password name 修改密码
sudo userdel name 删除用户（需要额外删除目录）

超级用户：root

sudo password root
su（switch user） root
sudo su
exit

登录root的桌面环境：

  1. 终端切换root身份
  2. 修改系统配置文件，gedit /etc/pam.d/gdm-autologin
    用#注释 auth required pam_succeed_if.so user !=root quiet_success
    gedit /etc/pam.d/gdm-password
    注释 auth requiredpam_succeed_if.so user!= root quiet success
  3. 重启

用户组：

groupadd group 创建组
useradd -m（modified） -g group name 创建用户入组
usermod -g boys name 修改用户入组
cat /etc/group 组
cat etc/password

可执行脚本

脚本：Script，一种解释执行的程序
Linux下常见的三种脚本程序：

  • Shell  *.sh
  • Perl   *.pl
  • Python  *.py

可以到脚本目录直接执行脚本，也可以到脚本解释器执行脚本

shell脚本

创建编辑
#！/bin/sh
echo "hello world"
添加可执行权限
chmod +x hello.sh
执行程序
./hello.sh

查看Ubuntu版本

cat /etc/lsb-release
cat /etc/os-release
uname -a

更新

apt-get update

docker

配置开机自启
sudo systemctl enable（关 闭换成diable） docker
查看 Docker 的运行状态：
sudo systemctl status docker
启动或停止docker命令
sudo service docker start 启动
sudo service docker stop 关闭

辅助

clear 清屏
history罗列历史命令

网络

ip -a/ifconfig查看ip
netstat -tunlp 查看端口
