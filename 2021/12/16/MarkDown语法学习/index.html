MarkDown语法学习

发表于 2021-12-16 更新于 2021-12-26 阅读次数：
本文字数： 2k 阅读时长 ≈ 2 分钟

通过hexo和gihub搭建的博客文章可以采用.md（Markdown）文件，我决定以边学边写的方式，通过此篇博客进行Markdown语法学习。

一，什么是MarkDown？

markdown是一种轻量化、纯文本的网页内容编辑格式，简单易学，可以与HTML混编，被简书、维基、github等采用。

Markdown编辑器
本人选择Visual Studio Code（官网免费），安装Markdown预览插件。

二，标题语法

通过添加#号进行标题分级
示例： 原markdown文件与实际效果对比
# 标题1

标题1

## 标题2

标题2

### 标题3

标题3

#### 标题4

标题4

##### 标题5

标题5

###### 标题6

标题6

*注：记得用一个空格在 # 和标题之间进行分隔。

三，段落和换行语法

用两个或多个空格进行换行。用空白一行进行分段
不过学到目前还不能段落缩进（开头空两格）

四，强调语法

  • 粗体

要加粗文本，请在单词或短语的前后各添加两个星号或下划线。
如:
**你好 ** –> 你好
__你好__ –> 你好

  • 斜体

单对星号或下划线包裹为斜体
_ 吃了吗您？_ –>吃了吗您？

  • 综合使用

粗体和斜体双重使用，用三对星号或下划线包裹

  • 删除线

可以通过在单词中心放置一条水平线来删除单词。结果看起来像这样。此功能使您可以指示某些单词是一个错误，要从文档中删除。若要删除单词，请在单词前后使用两个波浪号~~
删掉，删掉

五，引用语法

可添加>至段落前表示引用，如

这不是厕所，这是洗手间

可用>>进行嵌套

xmz说：

其他还有:
1.>## abc

abc

2.> - efg

  • efg

六,列表语法

  • 有序列表

请在每个列表项前添加数字并紧跟一个英文句点。数字不必按数学顺序排列，但是列表应当以数字 1 起始，记得空格。

  1. 我先学C语言
  2. 再学高数
  3. 要不先打会儿王者荣耀吧
  4. 我靠，打到十一点了，还学什么习呀

七，代码语法

将其包裹在` (键盘的Esc下面那个）中
如 #include<stdio.h>
或者也可以使用每一行缩进至少四个空格或一个制表符创建代码块 

int main(){
  int i=1;
  int sum=0;
  sum =(i++) + (++i);
  printf("%d",sum); 
  return =0;
}

八，分隔线

要创建分隔线，请在单独一行上使用三个或多个星号 (***)、破折号 (—) 或下划线 (___) ，并且不能包含其他内容。

九，链接与插入语法

  • 链接网址

链接文本放在中括号内，链接地址放在后面的括号中，链接title可选。
超链接Markdown语法代码：[超链接显示名](超链接地址 “超链接title”)
效果：超链接显示名

注：记得用英文括号

  • 图片

同类似于插入网址链接，写成![图片alt](图片链接 “图片title”)，但是，如果是使用hexo，需要进行一些改变；
可参考知乎hexo博客如何插入图片

十，转义字符语法

可用反斜杠进行转义字符的输出，

十一，内嵌html标签

于 Markdown 涵盖范围之外的标签，都可以直接在文件里面用 HTML 本身。如需使用 HTML，不需要额外标注这是 HTML 或是 Markdown，只需 HTML 标签添加到 Markdown 文本中即可。

十二，表格语法

使用三个或多个连字符（—）创建每列的标题，并使用管道（|）分隔每列。您可以选择在表的任一端添加管道。 

| Syntax      | Description |  
| ----------- | ----------- |  
| Header      | Title       |  
| Paragraph   | Text        |
Syntax Description
Header Title
Paragraph Text

Tip: 使用连字符和管道创建表可能很麻烦。为了加快该过程，请尝试使用Markdown Tables Generator。使用图形界面构建表，然后将生成的Markdown格式的文本复制到文件中。

十二，代码高亮

许多Markdown处理器都支持受围栏代码块的语法突出显示。使用此功能，您可以为编写代码的任何语言添加颜色突出显示。要添加语法突出显示，请在受防护的代码块之前的反引号旁边指定一种语言 

{
  "firstName": "John",
  "lastName": "Smith",
  "age": 25
}

十三，任务列表语法

任务列表使您可以创建带有复选框的项目列表。在支持任务列表的Markdown应用程序中，复选框将显示在内容旁边。要创建任务列表，请在任务列表项之前添加破折号-和方括号[ ]，并在[ ]前面加上空格。要选择一个复选框，请在方括号[x]之间添加 x 。
