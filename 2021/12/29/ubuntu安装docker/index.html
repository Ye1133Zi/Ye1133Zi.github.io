ubuntu安装docker

发表于 2021-12-29 更新于 2021-12-30 阅读次数：
本文字数： 804 阅读时长 ≈ 1 分钟

在学习docker的安装时，由于教程中的系统是centos，而我更熟悉ubuntu，所以我需要另外去找在Ubuntu系统下安装docker的方法

设为管理员模式
sudo su
cd
更新
apt-get update
下载依赖
apt-get install apt-transport-https ca-certificates curl gnupg-agent

software-properties-common
添加密钥
curl -fsSL https://download.docker.com/linux/ubuntu/gpg |apt-key add
验证
apt-key fingerprint 0EBFCD88
添加仓库
sudo add-apt-repository \ "deb [arch=amd64] https://download.docker.com/linux/ubuntu \ $(lsb_release -cs) \ stable"
更新
sudo apt-get update
安装docker
sudo apt-get install docker-ce docker-ce-cli containerd.io
检验安装
sudo docker run hello-world
配置非root用户使用docker
groupadd docker
sudo usermod -aG docker （用户名）
使配置生效(建议还是注销)
newgrp docker
检验
docker run hello-world
配置开机自启
sudo systemctl enable（关 闭换成diable） docker
查看 Docker 的运行状态：
sudo systemctl status docker
启动或停止docker命令
sudo service docker start 启动
sudo service docker stop 关闭
