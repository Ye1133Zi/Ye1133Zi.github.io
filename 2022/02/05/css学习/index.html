    布局

    文档流

    层式结构的最底层称为文档流

    元素在文档流中的特点

  • 块元素

    • 独占一行
    • 默认宽度是父元素的全部
    • 默认高度被子元素撑开

  • 行内元素

    • 不会独占一行
    • 不能容纳时换行

盒子模型

组成部分

  • 内容区（content）
  • 边框（border)
  • 内边距（padding）
  • 外边距（margin）

    边框

  • border-width
    边框宽度，顺序为上右下左，对称可省略
  • border-top/right/botttom/left-width/sytle/
  • border-color，默认黑色
  • border-style
    • solid,实线
    • dotted，点状虚线
    • dashed ，虚线
    • double，双线
    • none，没有
  • border一套全齐

    内边距

    内容区和边界之间的距离(会影响盒子大小，默认采取背景颜色)
  • padding-top/right/…
  • padding一套齐全

    外边距

    margin-top/bottom
    设置左、上默认移动元素，设置下、右移动其他元素

    水平布局与垂直布局

    垂直溢出overflow（-x/y）：visiable，hidden，scoll，auto

    行内元素的盒模型

    行内元素垂直边距不推挤，宽高不影响
    display：inline（行内元素），block（块元素），inline-block（设置宽高，不独占一行），table（表格），none（不在页面中显示）
    visibilty：visiable，hidden

    默认样式

    重置样式表

    盒子样式

    box-sizing属性

    设置box-sizing属性height和width改变盒子大小变化方式
  • content-box默认值，高宽设置内容区
  • border-box 高宽设置为整个盒子大小

    轮廓阴影和圆角

  • outline：设置轮廓，相对于不会挤元素的边框
  • box-shadow：设置元素阴影效果
    • 偏移量x
    • 偏移量y
    • 模糊效果（也用px）
    • color（一般会用rgba设置透明）
  • border-（top）-（left）-radius：设置圆角
    • 如果指定两个值为椭圆弧

      浮动

  • float特点
    元素设置浮动以后会完全从文档中脱离
    • 设置浮动以后，元素会向父元素中可占据的最左或最右移动
    • 浮动元素默认不会从父元素中溢出
    • 浮动元素向左或向右时，发生碰撞会停止
    • 浮动元素不会盖住文字，可以设置文字环绕图片效果
  • 脱离文档流
    • 块元素宽高会被内容撑开
    • 行内元素变成块元素
  • float值
    • none，默认值
    • left
    • right

      高度塌陷和BFC

      BFC块级格式化区域
  1. 开启BFC的元素不会被浮动元素所覆盖
  2. 开启BFC子元素和父元素外边距不会重叠
  3. 开启BFC的元素可以包含浮动的子元素

    开启BFC

  • 设置元素浮动
  • 将元素设置为行内块元素
  • 将元素的overflow设置为非visible（副作用小，hidden和auto）

    clear属性

  • 清除浮动元素对当前元素的影响
    属性值
  1. left
  2. right
  3. both（清除最大）
    原理
  • 设置浮动以后，浏览器自动为其添加一个外边距

    使用clear解决塌陷问题

    ：：after{
    content：’’;
    display：block;
    clear:both;

    解决外边距折叠

    ：：before{
    content：’’;
    display:table;
    }

    clearfix

    .clearfix::before , .clearfix::after{
    content:’’;
    display:table;
    clear:both;
    }

    定位

    position

    可选值
  • static 默认值，没开启定位
  • relative 开启相对定位
  • absolute 开启绝对定位
  • fixed 固定定位
  • sticky 开启粘滞定位

    相对定位

    相对定位不会脱离文档流

偏移量

  • top
  • bottom
  • right
  • left

    绝对定位

  • 会从文档流中脱离
  • 会改变元素性质，行内变成块
  • 相对于包含块进行定位

    包含块

  • 正常情况下
    • 包含块是离当前元素最近的祖先块元素
  • 开启定位
    • 只要开启定位，最近为包含块
    • 都没开启就是根元素

      固定定位

      属于绝对定位
      永远参照视口

      粘滞定位

      层级

      对于开启定位的元素，可以通过z-index属性指定元素层级
