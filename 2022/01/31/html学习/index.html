七,语义化标签

注意点：

  1. 一般情况下网页下只有一个<h1>,<h4>-<h6>很少用
  2. 在页面中独占一行的元素叫做快元素（block element），反之称为行内元素

行标签：

1.<p>标签中不能放块标签
2. <hgroup>将标题分组
3. <em>表示语音语调的加重
4. <strong>表示强调
5. <blockquote>表示长引用，<q>表示短引用
6. <br>换行
7. <span> 在网页中选中文字

块标签:

  1. <header>,<main>,<footer>
  2. <niv>导航
  3. <aside>和主体相关的其他内容
  4. <article>文章
  5. <section>表示独立的区块
  6. <div>一个区块

列表:

列表分类：

1.有序列表
2.无序列表
3.定义列表

使用<ol>标签创建有序列表
使用<ul>标签创建无序列表

超链接

使用<a>标签定义超链接
使用href属性给出链接地址
如

 <a href="http://ye1133zi.github.io">叶子的博客</a>

相对路径
./ 或 ../(类linux)

使用target属性（_self,_blank)选择新页面or当前页面打开超链接

回到顶部

<a href="#">回到顶部</a>

标记

使用id属性（锚点）标记标签，从而实现

<a href="#bottom">到底部</a>
...
<a id = "bottom">底部</a>

无效果的超链接

<a href="javascript:;">随便点</a>

图片标签

使用<img>自结束标签
src属性指定图片
alt添加描述，默认不会显示，有些浏览器在图片加载失败时显示
width和height属性设置大小

图片格式

  1. jpeg（jpg）
    • 支持颜色比较丰富，不支持透明效果和动图
    • 用来显示照片
  2. gif
    • 支持颜色比较少，支持简单透明和动图
    • 用来显示颜色单一的图片，动图
  3. png
    • 支持颜色丰富，支持复杂透明，不支持动图（？）
    • 适合网页
  4. webp
    • 谷歌新推出的专门用来表示网页中的图片的一种格式
    • 效果和大小都比较好，但是兼容性差
  5. base64
    • 将图片使用base64进行编码，将图片转化为字符
    • 和网页一起加载

音频，视频播放

音频

使用<audio>标签添加音频
添加controls属性让用户控制播放
添加autoplay属性设置自动播放，但是大部分浏览器不允许
添加loop属性控制循环播放
使用soure指定文件(多路径加播放失败后的提示)

    <audio controls>
        <!-- 对不起，宁德的浏览器不支持播放音频，请宁升级浏览器 -->
        <source src="./source/audio.mp3">
        <source src="./source/audio.ogg">
        <embed src=".source/audio.mpp3type="audio/mp3" wdith=300 height=100>

使用/标签为老浏览器添加音视频（需要指定width和height）
