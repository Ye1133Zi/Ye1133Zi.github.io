解决远程游戏语音回音：VB-Audio+windows录音侦听

发表于 2025-03-08 阅读次数：
本文字数： 670 阅读时长 ≈ 1 分钟

用免费的VB-Audio+windows录音侦听巧妙解决远程控制游戏时的语音回音问题

前情背景：

上周和好友使用网易uu远程控制，同时挂着微信语音，朋友那边出现了回音：远程控制软件默认会捕获系统所有声音，包括正在输出的朋友语音，又输出回朋友的耳朵。

初步解决方法：

第一步：被控端分出两个扬声器，一个只传游戏声音给远程控制软件，一个只传对方语言，这会导致被控端需要同时听两个扬声器（带两个耳机）

VB-Audio原理

VB-Audio一个虚拟声卡驱动，它的作用是让你的扬声器（电脑声音）变成麦克风（语音输入），而windows正好可以侦听一个麦克风给一个扬声器

最终方案

步骤1：安装虚拟声卡

  1. 前往VB-Audio官网下载VB-CABLE Driver
  2. 安装成功后，在Windows声音设置中会出现CABLE Input（虚拟麦克风）和CABLE Output（虚拟扬声器）

步骤2：配置音频输出分流

  1. 设置默认播放设备：右键系统托盘音量图标 → 打开声音设置 → 将默认播放设备设为CABLE Output，此时系统所有声音（包括游戏音效、朋友语音）会输出到虚拟声卡
  2. 单独指定语音软件输出：打开微信/Discord等语音软件 → 进入音频设置 → 将扬声器手动改为物理耳机/扬声器。原理：强制语音软件的音频直连真实耳机，与虚拟声卡隔离s
  3. 步骤3：启用Windows静音侦听打开控制面板 → 硬件和声音 → 声音 → 进入录制选项卡右键CABLE Input → 选择属性 → 进入侦听选项卡勾选侦听此设备。关键点：此操作会将虚拟声卡的声音“复制”到真实耳机，但由于语音软件已直连真实耳机，实际不会产生重复播放
