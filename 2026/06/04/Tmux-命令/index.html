3. 会话 Session

命令 / 快捷键 作用
tmux new -s work 新建名为 work 的会话
tmux ls 查看所有会话
tmux a 回到最近的会话
tmux a -t work 回到指定会话
prefix + d 脱离会话，进程继续跑
prefix + s 查看会话列表
prefix + $ 重命名当前会话

4. 窗口 Window

窗口类似浏览器标签页。

快捷键 作用
prefix + c 新建窗口
prefix + , 重命名窗口
prefix + & 关闭窗口
prefix + n 下一个窗口
prefix + p 上一个窗口
prefix + 0~9 跳到指定编号窗口
prefix + w 查看窗口列表

5. 面板 Pane

面板就是分屏。

快捷键 作用
prefix + % 左右分屏
prefix + " 上下分屏
prefix + x 关闭当前面板
prefix + z 当前面板全屏 / 还原
prefix + ! 当前面板转成独立窗口
prefix + q 显示面板编号
prefix + 方向键 切换面板
prefix + space 切换布局
